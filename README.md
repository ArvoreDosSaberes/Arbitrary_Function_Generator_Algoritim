## Visão geral

Este repositório contém um gerador de funções arbitrárias (AFG) com interface gráfica, voltado para criar e exportar formas de onda.

## Estrutura do projeto

- **`afg_gui.py`**: aplicação principal (GUI)
- **`waveform.txt`**: exemplo de arquivo de forma de onda

## Requisitos

- Python 3.10+ (recomendado)

## Como executar

1. (Opcional) Crie e ative um ambiente virtual.
2. Execute a aplicação:

```bash
python3 afg_gui.py
```
